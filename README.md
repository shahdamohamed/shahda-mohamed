# Shahda Mohamed Portfolio

A personal portfolio website for Shahda Mohamed, a backend developer based in Cairo, Egypt. The site presents Shahda's profile, skills, selected projects, CV download, and contact links in a clean API-inspired interface.

## Overview

This is a static HTML/CSS/JavaScript portfolio. It does not require a build step, backend server, or package installation.
