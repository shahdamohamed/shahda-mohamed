About me

I'm a backend developer with a B.Sc. in Computer Science from Benha National University, focused on designing and building reliable, well-structured APIs and data systems. My core stack is Python with Django and Django REST Framework, and I also build lightweight, high-performance services with FastAPI.

I think about a backend service end to end: how the data is modeled, how the API is shaped, and how it's secured — with maintainability always in mind, not just whether it works.

I work across the full backend lifecycle — from modeling data and designing clean, secure APIs, to building the authentication layer that protects them. I care as much about long-term maintainability as I do about a feature simply working.
